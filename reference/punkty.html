Punkt¶

class Punkt(id)[źródło]¶

Klasa opisująca Punkt porządku dziennego dla określonego posiedzenia.

Parametry:id (int) – Numer id punktu
classmethod lista()¶

Zwraca listę obietków typu Punkt dla wszystkich posiedzeń sejmu

id¶

Numer id punktu porządku dziennego. <int>

info[źródło]¶

Obiekt zawierający informacje o punkcie porządku dziennego. Posiada następujace atrybuty:

nr¶

Numer porządkowy punktu. <int>

tytul¶

Tytuł punktu. <unicode>

posiedzenie_id¶

Numer id posiedzenia. <int>

posiedzenie¶

Obiekt typu Posiedzenie o id = posiedzenie_id.

typ_id¶

Typ punktu. <int>

druk_id¶

Numer id druku, będącego przedmiotem punktu. <int>

druk¶

Obiekt typu Druk o id = druk_id, będący przedmiotem punktu.

druk_akcja_id¶

Typ podjętych akcji dotyczących druku. <int>

glosowanie_id¶

Numer id głosowania nad drukiem. <int>

glosowanie¶

Obiekt typu Glosowanie o id = glosowanie_id, będący głosowaniem nad punktem.

rozpatrywania¶

Lista obiektów typu Rozpatrywanie dla danego punktu porządku dziennego

druki¶

Lista obiektów typu Druk związanych z punktem porządku dziennego
