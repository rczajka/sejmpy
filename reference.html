Struktura¶

sejmometr.get_data(url)¶

Funkcja pobierająca dane z Sejmometr API. Jeżeli w systemie zainstalowana jest biblioteka httplib2 funkcja ta jest równa get_data_httplib2(), jeżeli httplib2 jest niedostępne to funkcja jest równa get_data_urllib2().

sejmometr.get_data_httplib2(url)¶

Funkcja wykorzystująca httplib2 do pobierania danych z Sejmometr API

sejmometr.get_data_urllib2(url)¶

Funkcja wykorzystująca urllib do pobierania danych z Sejmometr API

class sejmometr.Punkt(id)¶

Klasa opisująca Punkt porządku dziennego dla określonego posiedzenia.

classmethod lista()¶

Zwraca listę obietków typu Punkt dla wszystkich posiedzeń sejmu

id¶

Numer id punktu porządku dziennego. <int>

info¶

Obiekt zawierający informacje o punkcie porządku dziennego. Posiada następujace atrybuty.

nr¶

Numer porządkowy punktu. <int>

tytul¶

Tytuł punktu. <unicode>

posiedzenie_id¶

Numer id posiedzenia. <int>

posiedzenie¶

Obiekt typu Posiedzenie o posiedzenie_id. <Posiedzenie>

typ_id¶

Typ punktu. <int>

druk_id¶

Numer id druku, będącego przedmiotem punktu. <int>

druk¶

Obiekt typu Druk o druk_id, będący przedmiotem punktu. <Druk>

druk_akcja_id¶

Typ podjętych akcji dotyczących druku. <int>

glosowanie_id¶

Numer id głosowania nad drukiem. <int>

glosowanie¶

druki¶
