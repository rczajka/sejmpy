sejmometr.get_data(url)¶

Funkcja pobierająca dane z Sejmometr API. Jeżeli w systemie zainstalowana jest biblioteka httplib2 funkcja ta jest równa get_data_httplib2(), jeżeli httplib2 jest niedostępne to funkcja jest równa get_data_urllib2().

sejmometr.get_data_httplib2(url)¶

Funkcja wykorzystująca httplib2 do pobierania danych z Sejmometr API

Funkcja wykorzystująca urllib2 do pobierania danych z Sejmometr API
